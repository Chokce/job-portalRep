3. Database Setup Instructions

üìã Step-by-Step Setup:

  1. Create .env file in the server folder with:
    DATABASE_URL=postgresql://username:password@localhost:5432/jobportal JWT_SECRET=your_secret_key_here PORT=4000
  2. Install PostgreSQL if not already installed
  3. Create database: createdb jobportal
  4. Run setup: npm run setup-db
  5. Run migrations: npm run migrate
  6. Start server: npm run dev
